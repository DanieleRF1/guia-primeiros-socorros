# Guia Digital Comunitário de Primeiros Socorros

Projeto desenvolvido para a disciplina de Desenvolvimento Frontend da UNISUAM.

## Objetivo

Criar um site com utilidade social contendo informações de primeiros socorros, contatos úteis e conteúdos educativos.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- GitHub
- GitHub Pages

## Funcionalidades

- Layout responsivo
- Menu interativo
- Busca dinâmica
- Contatos de emergência
- Formulário com validação

## Estrutura do Projeto

- Página inicial
- Página de engasgo
- Página de queimaduras
- Página de emergência
- Formulário de contato

## Publicação

O projeto foi publicado utilizando GitHub Pages.
